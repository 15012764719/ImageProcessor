Filter

Applies a filter to the current image. Use the MatrixFilters class to assign the correct filter

{% highlight c# %} public ImageFactory Filter(IMatrixFilter matrixFilter) {% endhighlight %}

Parameters

matrixFilter
The IMatrixFilter to add to the image. Found in the MatrixFilters class

Available filters are:

  • BlackWhite
  • Comic
  • Gotham
  • GreyScale
  • HiSatch
  • Invert
  • Lomograph
  • LoSatch
  • Polaroid
  • Sepia
